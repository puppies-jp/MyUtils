## <a name=tag>id/class/要素 を指定した書き方について</a>

```css
.<class名>{}
/*
.nameClass {color: red;}
これもできる
<タグ名>.<class名> {color : black}
*/

#<id名> {}
/* 
#nameId {color: blue;}
<タグ名>#<id名> {color : black}
*/

<タグ名> {}
/*
div { color: yellow;}
*/

/* 複数指定　は以下(タグ、id,classなんでもOKらしい) */
◯◯,xx,△△ {

}
```
